## 🌐 TAB 1: 시장 (📡 레이더)

시장 전체 분위기를 한눈에 파악하는 대시보드입니다.

### 지수 카드 (클릭 → 미니차트)
S&P 500, NASDAQ, KOSPI, VIX, USD/KRW, 미국 10Y 금리, 금 — 7개 지수 표시.  
**아무 카드나 클릭하면 30일 추이 미니차트가 펼쳐집니다.** 다시 클릭하면 접힘.

### 섹터 RS 히트맵
- 미국/한국 섹터별 상대강도(RS)를 색상으로 표시
- **초록**: 시장 대비 강세 / **빨강**: 시장 대비 약세
- 1주 또는 1개월 기간 선택 가능
- 섹터 클릭 → 구성종목 표시 → ★ 버튼으로 관심종목 추가

### 상승/하락 비율
한국·미국 시장의 상승 종목 비율을 바 차트로 표시. 50% 이상이면 시장 전반적 강세.

### 시장 기회 점수 (US / KR 분리)

**미국 점수 산출 기준:**
| 요소 | 점수 영향 |
|------|-----------|
| VIX < 15 | +22점 |
| VIX 15~20 | +12점 |
| VIX 25~30 | -12점 |
| VIX > 30 | -22점 |
| S&P 3일 > +2% | +18점 |
| S&P 3일 > 0% | +8점 |
| S&P 3일 < -2% | -18점 |
| 미국 강세 섹터 1개당 | +3점 |

**한국 점수 산출 기준:**
| 요소 | 점수 영향 |
|------|-----------|
| KOSPI 3일 > +2% | +22점 |
| KOSPI 3일 > 0% | +10점 |
| KOSPI 3일 < -2% | -22점 |
| 한국 강세 섹터 1개당 | +4점 |

- **70점 이상 → HIGH**: 적극 매수 환경
- **45~69점 → MID**: 선별 매수
- **44점 이하 → LOW**: 관망 또는 현금 비중 확대

## 🔍 TAB 2: 발굴

관심종목 중 매수 조건을 충족하는 종목을 필터링합니다.

### 알파 스크리너 점수 산출
| 조건 | 점수 |
|------|------|
| RS(상대강도) > +5%p | +35점 |
| RS > +2%p | +25점 |
| RS > 0%p | +15점 |
| 일목구름 위 | +10점 |
| ST 3/3 (트리플 슈퍼트렌드 매수) | +15점 |
| ST 2/3 | +8점 |
| 골든크로스 (3일/10일) | +10점 |
| 거래량 200%+ (20일 대비) | +10점 |
| 거래량 150%+ | +5점 |
| 52주 신고가 근접 | +5점 |

> **RS(상대강도)란?** 해당 종목의 5일 수익률 - 시장지수 5일 수익률.  
> 예: 삼성전자 5일 +3% / KOSPI 5일 +1% → RS = +2%p  
> 시장보다 더 오르는 종목 = 수급이 집중되는 종목.

### 필터 조건
- **시장**: 전체 / 한국 / 미국
- **슈퍼트렌드**: 전체 / 1개+ / 2개+ / 풀(3/3)
- **일목구름**: 전체 / 접근+ / 구름 위
- **RS 강도**: 슬라이더로 최소 RS 설정
- **거래대금**: 슬라이더로 최소 거래량 비율 설정

### 테이블 첫줄
각 나라 섹션 상단에 **대표지수 현재 등락률**이 표시됩니다.  
예: `🇰🇷 한국 3개 — KOSPI +0.85% 3D +1.2%`

### 백테스트 (🔬)
보유 종목들의 과거 데이터로 전략을 시뮬레이션합니다.

**설정 가능 조건:**
- ST 3/3 / 구름 위 / MACD 골든크로스 / 거래량 급증 / 52주 신고가
- 손절 % (3~25%) / 익절 % (5~50%)

**결과 지표:**
| 지표 | 의미 |
|------|------|
| 승률 | 수익 거래 비율 |
| 평균 손익 | 거래당 평균 수익률 |
| Profit Factor | 총이익 ÷ 총손실 (1.5+ 양호) |

### 비교뷰 (⚖️)
관심종목(★)끼리 RS, 거래량, 일목구름 상태를 테이블로 비교합니다.

## 📊 TAB 3: 차트

선택 종목의 기술적 분석 차트와 진입 판단 도구입니다.

### 진입 평점 (참고용, 강제 제한 없음)
| 항목 | 최대 점수 | 기준 |
|------|-----------|------|
| 일목구름 위 | 15점 | 가격 > 구름 상단 |
| 트리플 ST | 25점 | 3개 SuperTrend 모두 매수 |
| MACD 크로스 | 15점 | MACD > Signal 및 양전 |
| RS 순위 | 25점 | 풀 데이터 기준 상위 % |
| RSI 모멘텀 | 10점 | RSI 60~80 강세구간 |
| 52주 신고가 | 10점 | 고점 95% 이상 |

- **S등급 (85+)**: 최적 진입 조건
- **A등급 (70~84)**: 양호
- **B등급 (55~69)**: 보통
- **C등급 (40~54)**: 주의 필요
- **D등급 (~39)**: 진입 비권장

### 차트 지표
| 지표 | 설명 |
|------|------|
| **SuperTrend 3중** | 초록=매수, 빨강=매도. 3개 모두 초록이면 강한 매수 신호 |
| **HMA 20** | 주황 점선. Hull Moving Average — 빠른 추세 |
| **MA 200** | 회색 점선. 장기 추세선 |
| **MACD** | 골든크로스(파랑>노랑) = 매수, 데드크로스 = 매도 |
| **RSI** | 70 이상 과매수, 30 이하 과매도 |
| **Squeeze TTM** | 🟡 압축 중 → 🔴 해제 시 방향 폭발 |
| **ADX** | 25+ = 추세 강함 (토글로 ON/OFF) |
| **OBV** | 거래량 균형 — 가격 상승 시 증가하면 건전한 상승 |
| **AVWAP** | 앵커드 VWAP — 최근 저점 기준 평균 단가 |

### 매물대 저항선 (🧱)
과거 고점이 겹치는 가격대를 자동 감지합니다. 이 가격 근처에서 매도 압력이 나올 수 있습니다.

### 매수 등록
체크리스트 5개 항목을 모두 체크하면 매수 버튼이 활성화됩니다.  
매수 시 **1차 정찰 금액(30%)**만 자동 투입되고, 보유중 탭으로 이동합니다.

## 📁 TAB 4: 추적

### 👁 관찰중
발굴탭에서 "추적" 버튼을 누른 종목들. 아직 매수하지 않고 지켜보는 단계입니다.
- 발굴 시점의 점수, RS, 신호가 기록됨
- "매수 전환" 버튼으로 보유중 탭으로 이동 가능

### 💼 보유중

#### 불타기 룰 (30/30/25/15)
| 단계 | 비중 | 조건 | 의미 |
|------|------|------|------|
| 1차 정찰 | 30% | 진입 시 | 소량으로 시장 반응 확인 |
| 2차 확인 | 30% | 평단가 +3% | 방향 확인 후 본격 투입 |
| 3차 추세 | 25% | 평단가 +5% | 추세 타고 추가 |
| 4차 가속 | 15% | 평단가 +8% | 마무리 물량 |

> 종목당 최대 투자금 = 총 투자금 × 종목당 최대 비중(%)  
> 예: 1,000만원 × 30% = 300만원 → 1차 90만 / 2차 90만 / 3차 75만 / 4차 45만

#### 손절 규칙
| 상태 | 기준 | 기본값 |
|------|------|--------|
| **초기 손절** | 매수가 기준 -N% | -10% |
| **트레일링** | 고점 기준 -N% | 고점 -8% |
| **전환 조건** | 수익 +N% 이상 달성 시 | +10% |

예: 10만원에 매수 → 초기 손절 9만원.  
12만원까지 오름(+20%) → 트레일링 전환 → 고점 12만 × (1-8%) = 11,040원이 새 손절.

#### 타임컷 ⏰
- **기본 14일** 보유 후 손익 **±3% 이내**면 "박스권 정체" 경고
- 주황 배너로 표시 + **원클릭 타임컷 청산** 버튼
- 리스크 설정에서 기간(5~30일)과 범위(±1~8%) 조절 가능

> 왜 타임컷? 움직이지 않는 종목에 자본을 묶어두면 기회비용 발생.  
> 빠르게 손절하고 다음 기회로 넘어가는 것이 추세추종의 핵심입니다.

#### 경고 배너 (우선순위 순)
1. 🚨 **손절선 근접** (2% 이내) — 빨간 배너 + 펄스 애니메이션
2. ⏰ **타임컷** (N일 정체) — 주황 배너 + 청산 버튼
3. ⚠️ **거래량 급감** (20일 평균 50% 이하) — 노란 배너
4. 🔥 **불타기 목표 도달** — 초록 배너

### ✅ 청산
- 모든 청산 기록이 테이블로 표시
- **📥 CSV 내보내기**: UTF-8 엑셀 호환 CSV 다운로드
- 종목, 매수가, 청산가, 손익%, 청산사유, 신호 포함

### 📈 통계

#### 에쿼티 커브
청산 기록 기반 누적 수익률 그래프. 우상향이면 전략이 작동하는 것.

#### 신호별 성과
어떤 매수 신호(ST3/3, 구름위, MACD↑ 등)가 높은 승률을 보이는지 분석.

#### OppScore별 성과
시장 점수가 HIGH/MID/LOW일 때 각각의 거래 승률을 비교.  
→ HIGH일 때만 매수하는 것이 유리한지 데이터로 확인.

#### 조건 조합 분석
"ST3/3 + 구름위" 같은 2개 이상 조건 조합의 승률을 보여줍니다.  
풀조합(3개+)의 승률이 단일 조건보다 높으면 전략이 검증된 것.

#### AI 성적 분석
`/api/analyze` 백엔드가 연결되어 있으면 Claude AI가 매매 패턴을 분석합니다.  
연결되지 않아도 앱은 정상 작동하며, "AI 분석 실패" 메시지만 표시됩니다.

| 기록 항목 | 설명 |

## ❓ FAQ

**Q: 뉴스 점수나 SNS 감성 점수 기능은?**  
A: 현재 v2.1에는 포함되어 있지 않습니다. 향후 API 연동 시 추가 가능합니다.

**Q: 데이터가 안 불러와져요**  
A: GitHub Actions의 `daily.yml`을 수동 실행(Run workflow)해보세요. 성공하면 `/data/stocks.json`이 생성됩니다.

**Q: AI 분석 버튼이 실패해요**  
A: 정상입니다. `/api/analyze` 백엔드가 없으면 에러 메시지만 뜨고 앱은 정상 동작합니다.

**Q: 종목을 추가했는데 차트가 시뮬레이션이에요**  
A: 실시간 데이터(candles)가 없는 종목은 시뮬레이션 차트를 생성합니다. "시뮬" 뱃지가 표시됩니다.

**Q: localStorage가 꽉 차면?**  
A: 관심종목, 포지션, 청산기록, 일지가 모두 브라우저 localStorage에 저장됩니다. 브라우저 데이터 삭제 시 초기화되므로, 중요한 기록은 CSV로 백업하세요.

## 📋 v2.1 추가 기능 상세

### 📐 피보나치 되돌림
차트탭 매물대 저항선 아래에 표시됩니다.
- **60일 기준** 고점~저점 사이의 38.2%, 50%, 61.8% 레벨
- 현재가가 특정 레벨 근처(±1%)면 하이라이트 표시
- 각 레벨에서 현재가 대비 상승/하락 % 표시

| 레벨 | 의미 |
|------|------|
| 38.2% | 약한 되돌림 — 강한 추세에서 1차 지지 |
| 50.0% | 중간 되돌림 — 가장 흔한 반등 지점 |
| 61.8% | 깊은 되돌림 — 황금비율, 여기서 반등 못 하면 추세 전환 |

### 💰 거래대금
보조지표에 거래대금(억/M) 추가. 거래량%와 함께 표시됩니다.
- **거래량%**: 20일 평균 대비 비율 (상대적 활발도)
- **거래대금**: 실제 거래된 금액 (절대적 유동성)
- 소형주 150%와 대형주 150%를 구분할 수 있습니다

### 🧮 매수 시뮬레이터 (풀세트)
차트탭에서 매수 전 시뮬레이션을 보여줍니다.

**ATR 일변동폭**: 이 종목이 하루 평균 몇 % 움직이는지
- 예: ATR ±2.3% → 하루에 2.3%씩 움직이는 종목
- 손절폭이 ATR보다 작으면 노이즈에 걸릴 수 있음 (주의)

**목표 도달 예상**: ATR 기준으로 목표가까지 약 며칠 걸리는지 추정
- 예: 목표 +10%, ATR 2.3% → ~4~5일 예상

**예상 손익 계산**:
| 시나리오 | 계산 |
|----------|------|
| 전량 손절 | 1차 매수금 × 손절% |
| 2R 목표 | 1차 매수금 × 손절% × 2 |
| 풀매수+목표 | 전체 투입금 × (목표가-현재가)/현재가 |

### 📊 청산 내역 보유일수
청산 테이블에 "보유" 컬럼 추가.
- 타임컷 청산은 주황색으로 표시
- 평균 보유일수를 보면 본인의 트레이딩 스타일 파악 가능
